PostEvent(ref UioHookEvent)

Posts a fake input event.

Declaration
public static extern void PostEvent(ref UioHookEvent e)
Parameters
Type Name Description
UioHookEvent e

The event to post.
Remarks

The instance of the event doesn't need all fields to have value. Only Type, Keyboard/Mouse/Wheel and Mask should be present.

The following table describes the specifics of simulating each event type.

Event typeDescription
HookEnabledEvents of this type are ignored.
HookDisabledEvents of this type are ignored.
KeyPressedOnly KeyCode is considered.
KeyReleasedOnly KeyCode is considered.
KeyTyped Not recommended to use since on some platforms events of this type are ignored, while on others they are not.
MousePressedOnly Button is considered.
MouseReleasedOnly Button is considered.
MouseClicked Not recommended to use since on some platforms events of this type are ignored, while on others they are not.
MouseMovedOnly X and Y are considered.
MouseDraggedNot recommended to use; instead, use MouseMoved with a modifier mask.
MouseWheel Only X, Y, Amount, and Rotation are considered.

Mouse wheel simulation is a little inconsistent across platforms, and not documented. View the source code of libuiohook for more details.

See Also
EventSimulator
